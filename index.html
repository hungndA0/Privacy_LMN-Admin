Privacy Policy

Product: LMN Admin - License Key Tool
Last updated: March 20, 2026

This Privacy Policy explains how the LMN Admin - License Key Tool Chrome extension handles information.

1. What this extension does

The extension is used by authorized administrators to generate signed license keys for Live Meeting Notes.

2. Data collection and transmission

The extension does not collect analytics and does not transmit personal data to developer-owned servers.

  • No tracking scripts are used.
  • No advertising SDK is used.
  • No sale of user data.

3. Local processing

Key generation and cryptographic operations are performed locally in the browser using built-in Web Crypto APIs.

4. Storage

The extension stores an encrypted private-key blob in Chrome storage sync for administrator convenience across signed-in Chrome profiles.

  • The private key is encrypted before storage.
  • The encryption password is provided by the administrator.

5. Chrome permissions used

  • storage: store encrypted extension data needed for operation.
  • tabs: open the extension page in a browser tab when the toolbar icon is clicked.

6. Third-party services

The extension does not require third-party API calls to generate license keys.

7. Changes to this policy

This policy may be updated from time to time. The latest version will be published at the same URL.
